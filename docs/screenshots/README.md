# Screenshot Placeholders

Add screenshots used by `README.public.md` with these filenames:

- `dashboard.png`
- `course-syllabus.png`
- `lesson-view.png`
- `admin-analytics.png`

Tips:
- Keep width around `1600px` for crisp GitHub rendering.
- Blur or remove any secrets/tokens before uploading.
- Prefer PNG for UI screenshots.
